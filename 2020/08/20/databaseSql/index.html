database/sql包

go语言 数据库 mysql
发布日期:   2020-08-20

database/sql包是go语言官方提供的sql连接和使用的包，用以操作SQL和类SQL数据库，但是官方没有提供驱动，需要自己手动安装

安装mysql驱动包

在终端输入以下命令,就能安装mysql的驱动包

E:\code\go\src\design>go get github.com/go-sql-driver/mysql
go: github.com/go-sql-driver/mysql upgrade => v1.5.0

导入相关包

在代码中导入database/sql包和驱动包,驱动包需要使用下划线导包的方式

import (

    "database/sql"
    _ "github.com/go-sql-driver/mysql"

)

连接mysql

sql.Open()函数可以连接数据库，第一个参数是驱动名称，第二个参数是连接数据库必须要用到信息,DB不是连接，只有当需要使用时才会创建连接，如果想立即验证连接，需要用Ping()方法

db,err:=sql.Open("mysql","user:password@tcp(127.0.0.1:3306)/database")
    if err !=nil{
        //fmt.Println("连接失败")
    }
    defer db.Close()
    err :=db.Ping()
    if err!=nil {
        fmt.Println("状态错误")
    }

DQL

使用数据库查询语言时，先用rows对象(sql.go中定义的结构体)接收db.Query()查询的结果，然后使用rows对象的next方法进行迭代,rows使用完也需要使用close方法关闭资源，把遍历到的数据存入变量使用rows.Scan()

var sql string="select user_id,user_name from user;"
    var(
        id string
        name string
    )
    rows,err:=db.Query(sql)
    if err!=nil {
        fmt.Println("查询失败")
    }
    defer rows.Close()
    for rows.Next() {
        err :=rows.Scan(&id,&name)
        if err!=nil{
            fmt.Println("查询失败")
        }
        fmt.Printf("%s-----%s\n",id,name)
    }

DML

在使用数据库操作语言的时候，使用db.Exec()方法可以执行insert、update、delete等语句

var sqlStr="update user set user_name=? where user_id=?"
     result,err:=db.Exec(sqlStr,"张三",5)
     if err!=nil{
         fmt.Println("跟新失败")
     }
     rows,err:=result.RowsAffected()
     if err!=nil{}
     fmt.Println(rows)

DDL

使用数据库定义语言的时候，同样是通过db.Exec()方法

var sqlStr="create table test(id int not null,info varchar(100) not null)"
    res,err:=db.Exec(sqlStr)
    if err!=nil {
        fmt.Println("err")
    }
    fmt.Println(res.RowsAffected())

预处理

普通SQL语句的执行：
  1. 客户端对SQL语句进行占位符替换的到完整的SQL语句
  2. 将完整的SQL语句发送到mysql服务端
  3. mysql服务端执行SQL语句并返回结果
预处理的sql：
  1. 将SQL语句分为命令部分和数据部分
  2. 先把命令部分发送给mysql服务端，mysql服务端进行SQL预处理
  3. 然后把数据部分发送给mysql服务端进行占位符替换
  4. mysql执行完整的SQL语句并返回结果
预处理的好处：
  1. 优化mysql重复执行sql的方法，可以提升服务器性能，提前编译之后，可以多次执行
  2. 可以避免SQL注入

        var sqlStr="select user_name from user where user_id<3"
    statement,err :=db.Prepare(sqlStr)
    if err!=nil{}
    defer statement.Close()
    rows,err:=statement.Query()
    var name string
    for rows.Next() {
        err:=rows.Scan(&name)
        if err!=nil{}
        fmt.Println(name)
    }
    rows.Close()
